## Backup script
This script quickly and easily creates an archive of your documents.
Furthermore, the script works in two ways (by *menu* or by *options*).

## Usage
	$ ./run.sh -h
	help option
		-s, coping data for backup
		-ar, create tar file
		-c, compress data
		-r, restor data
